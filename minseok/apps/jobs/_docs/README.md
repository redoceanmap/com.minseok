# 래그테일러(Ragtailor)를 찾아주셔서 진심으로 감사드립니다

안녕하세요, **래그테일러(Ragtailor)**의 대표 **Rex Park**입니다.

저희 홈페이지를 방문해 주시고 저희가 걸어가는 여정에 관심을 가져주셔서 깊은 감사의 말씀을 전합니다.

---

## 💡 Our Mission: Vibe Coding & Hexagonal Architecture

래그테일러는 AI 시대를 살아가는 비즈니스 파트너와 개발자들을 위해 **가장 직관적이고 견고한 AI 애플리케이션 생태계**를 고민합니다. 

우리는 복잡하고 경직된 기존의 개발 패러다임을 넘어, AI와 인간이 유기적으로 호환되는 **Vibe Coding 방법론**을 지향하며, 동시에 변경에 유연하고 확장성이 뛰어난 **핵사고날 아키텍처(Hexagonal Architecture)** 중심의 시스템을 설계합니다.

* **RAG(Retrieval-Augmented Generation) 최적화:** 파편화된 기업의 비즈니스 데이터를 LLM과 완벽하게 융합합니다.
* **FastAPI & Next.js 기반 빌딩:** 가볍고 빠른 백엔드 아키텍처와 직관적인 프론트엔드를 모노레포 구조로 안전하게 통합 관리합니다.
* **인간 중심의 도메인 설계:** 복잡한 비즈니스 actor 구조를 가장 명확하고 상식적인 언어로 정립하여 시스템을 구축합니다.
